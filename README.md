# Apple-Social-Media-Analytics-using-Excel
# Objective:
Build a Social Media Intelligence Solution in Excel that consolidates Apple’s raw data into actionable business insights. 
To make data-driven decisions for improving campaign performance, engagement, and platform strategy. 
# Tools used:
Microsoft Excel (Pivot Tables, Charts, Filters, Formulas).
# Dataset:
Posts, Engagement summary, Campaign Metadata
# Tasks performed:

Analyzed Apple’s social media engagement data to identify trends in likes, shares, and comments.

Cleaned and organized raw data using Excel formulas, filters, and pivot tables.

Created visual reports and charts to highlight audience behavior and content performance.
# Key Insights:

Based on the average engagement rate, Twitter outperformed all other platforms.

#iPhone16Launch has the highest average clicks that is 1811 and lowest is #MacBookProM3 with 1289

#HealthTech (14.38%) performs best among all the hashtags baased on their Average Engagement rate

“Among all platforms, LinkedIn recorded the highest total engagement, followed by Twitter, YouTube, and Instagram.”

Videos (13.93%) have the highest overall average engagement across platforms, followed by Carousel, Images and Tweets.
